HSE Leitner Pro v8
==================

A professional multilingual HSE vocabulary trainer for GitHub Pages.

Main features
- English interface by default
- Switchable interface: English, German and Persian
- 1,015 unique HSE terms without artificial prefixes or duplicate phrase cards
- German, English and Persian terminology
- Two practical German examples with Persian translations
- Leitner learning system
- Clickable dictionary categories
- Expandable term details
- Search across all three languages and sources
- Personal term creation with manual entry or secure AI endpoint
- Local backup and restore
- PWA and offline support

GitHub Pages
Upload the extracted files directly to the repository root. Do not upload only the ZIP file and do not place index.html inside an additional folder.

Required root files include:
index.html
vocabulary.json
manifest.webmanifest
sw.js
.nojekyll

Then enable GitHub Pages from the main branch and /(root).


Version 8 adds concise English and German definitions for all 1,015 core terms. Definitions are shown before practical examples in cards and dictionary details.
